Order
=====


Fields
------

orderId
^^^^^^^

      Unique system generated string identifier of the drink.

      **Type**: string

      **Required**: Yes

drink
^^^^^

      Name of the drink

      **Type**: string

      **Required**: Yes

additions
^^^^^^^^^

      List of additions (flavors) to be included in the drink

      **Type**: list(string)

      **Required**: Yes

cost
^^^^

      Cost of the drink in USD

      **Type**: double

      **Required**: Yes

next
^^^^

      A URL pointing to the next resource in the workflow

      **Type**: href

      **Required**: Yes
